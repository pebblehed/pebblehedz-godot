# Pebble Hedz Experience Direction

**Status:** Authoritative Foundation  
**Project:** Pebble Hedz — Godot Edition  
**Owner:** PixelSync  
**Last reviewed:** 2026-07-17

## Experience Goal

Pebble Hedz should feel premium, calm, tactile, and compulsively replayable.

The experience is built from contrast:

```text
calm atmosphere
+
run tension
+
small moments of surprise and rescue
```

## ASMR Direction

Audio should support concentration, tactile satisfaction, and emotional feedback.

Priorities:

- real water character;
- gentle ripple and shore ambience;
- clear but restrained pebble-water impacts;
- subtle environmental interaction sounds;
- satisfying reward cues that do not become noisy or arcade-heavy.

Avoid:

- synthetic splash spam;
- constant reward noise;
- dense overlapping effects;
- audio that competes with the water;
- excessive UI sound.

## Visual Direction

The visual world should remain readable and uncluttered.

Water is the visual anchor.

Environmental elements should make the world feel alive while preserving space and calm.

Lily pads should:

- visibly belong to the water;
- follow the local surface height;
- react subtly to wave movement;
- be recognisable at mobile scale;
- provide a clear interaction moment when they affect a run.

## Emotional Rhythm

A strong run should create an emotional sequence similar to:

```text
anticipation before launch
→ satisfaction on good skips
→ tension as energy fades
→ hope when an environmental opportunity appears
→ excitement when the run is rescued
→ disappointment at failure
→ immediate desire to retry
```

## Premium Restraint

Premium does not mean adding more effects.

Premium means:

- responsive interactions;
- strong timing;
- clean visual hierarchy;
- considered sound;
- coherent motion;
- meaningful feedback;
- absence of unnecessary clutter.

## Mobile-First Direction

Every feature should remain suitable for short sessions and touch interaction.

Protect:

- immediate startup;
- simple input;
- readable scale;
- strong performance;
- instant retry;
- low cognitive overhead.
